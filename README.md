## Python web scraping with Selenium
Scrape and extract job data using selenium and store in Pandas DataFrame.

### Run the app locally in VS Code

1. Install Python on local machine
2. Install Python Extension on VS Code
3. Select Python interpreter, CRTL + Shift + P, Python: Select iInterpreter
4. Create a virtual environment for your packages, CRTL + Shift + P, Python: Create Environment
5. Select `Venv`, the latest available version of Python and then run pip install -r requirements.txt to install packages inside virtual environment.
6. Press **F5** to run your application!
